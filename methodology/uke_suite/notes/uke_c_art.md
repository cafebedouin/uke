## §2: PATTERN RECOGNITION (FLAG GENERATION - IMAGE)

### §2.1: FLAG DEFINITIONS (IMAGE-ADAPTED)

```markdown
[FLAG: SYMBOL_CANDIDATE]
Threshold:
- Visual element (shape, color, motif) appears ≥3 times (§1.7-10)
- AND visual treatment or spatial context shifts ≥1 time
- AND not explained as decorative repetition (e.g., wallpaper pattern)

Evidence Required:
- List element + all locations (map on work)
- Describe transformation (scale, color, clarity, position changes)

[FLAG: ABSENCE_STRUCTURAL]
Threshold:
- Genre/tradition convention visually present in 80%+ of comparison works
- AND explicitly missing from this work (§1.11)
- OR significant negative space with compositional impact (§1.12)

Evidence Required:
- Name expected element
- Cite 3+ comparison works where it appears

[FLAG: META_GESTURE]
Threshold:
- Work explicitly references its own making (§1.22)
- OR breaks visual frame/illusion (§1.23)
- OR material performs meaning (§1.24)

Evidence Required:
- Location + description or photograph

[FLAG: REGISTER_COLLISION]
Threshold:
- Two+ distinct visual styles/systems appear within single work (§1.14, §1.16, §1.18)
- AND shift is not traditional contrast (figure/ground difference)
- Measured by: Style clash (photorealism + abstraction)
  OR spatial contradiction (flat + illusionistic in same plane)
  OR material clash (refined + crude in same work)

Evidence Required:
- Location showing collision
- Identify each register

[FLAG: CONSTRAINT_FORMAL]
Threshold:
- Work adheres to externally-defined rule set (e.g., monochrome, grid system, single material)
- OR self-imposed constraint mentioned in statement/documentation
- Measured by: Structural consistency ≥90% (§1.7-9)

Evidence Required:
- Name constraint
- Show adherence across work

[FLAG: ALLUSION_NETWORK]
Threshold:
- Direct visual reference to ≥2 external works/traditions (§1.24)
- OR sustained stylistic appropriation (≥30% of work mimicking specific source)

Evidence Required:
- Location + description of reference
- Identify sources

[FLAG: INTENSITY_HIGH]
Threshold:
- Visual density: >70% of surface filled with marks/detail
- OR color saturation: >5 high-chroma colors at full intensity
- OR textural complexity: >3 distinct surface treatments simultaneously visible

Evidence Required:
- Photograph or description of dense area
- Provide measurements

[FLAG: INTENSITY_LOW]
Threshold:
- Visual sparseness: <30% of surface marked/occupied
- OR monochrome: Single hue or value range <15% variation
- OR minimal intervention: Raw material dominant, <20% artist mark

Evidence Required:
- Photograph or description of sparse area
- Provide measurements

[FLAG: CONTEXT_EXPLICIT]
Threshold:
- Work depicts specific historical event, person, or place (visually identifiable)
- OR includes text naming context
- OR genre conventions clearly respond to cultural moment (§1.11, §1.24)

Evidence Required:
- Identify depicted subject or quote text
- Provide historical context

[FLAG: FAILURE_VISIBLE]
Threshold:
- Genre/tradition promise broken (§1.2 signals X, work delivers Y)
- OR technical error repeated ≥3 times (structural flaws, unintended damage, execution problems)
- OR claimed intent (statement/documentation) contradicts visible result

Evidence Required:
- State promise
- Photograph failure instances
```

## Critical Insight: What Static Visual Art Proves

**Image has no temporal unfolding AND no language**, yet the same flags work because:

1. **Repetition still operates spatially** (across surface instead of across time)
2. **Transformation tracks** (not "becoming" but variation in simultaneity)
3. **Absence functions compositionally** (negative space = structural gap)
4. **Registers collide** (style/technique/material instead of tone/tempo)

The hostile observer test **still works**: Two critics could independently map "red circle appears 4× with decreasing opacity and scale variation" without agreeing on meaning.
